## 功能特性

- 🎬 **视频列表爬取**: 支持多种分类，获取指定数量的视频信息
- 🖼️ **自动打码**: 自动对视频封面进行马赛克处理，确保内容安全
- 📱 **即时返回**: 直接在聊天中返回结果，无需文件操作
- ⚙️ **配置管理**: 支持自定义爬虫参数和打码程度
- 🔒 **安全合规**: 内置安全机制，适合平台使用

## 安装要求

- Python 3.7+
- AstrBot框架
- 依赖包（见requirements.txt）

## 安装步骤

1. 将插件文件复制到AstrBot插件目录
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 在AstrBot中启用插件

## 使用方法

### 基本命令

#### 爬取视频列表
```
/91porn [分类] [数量]
```

参数说明：
- 分类：rf(热门), mv(最新), vd(视频)等，默认为rf
- 数量：要获取的视频数量，默认为5，最大10

示例：
```
/91porn rf 5    # 获取热门分类5个视频
/91porn mv 3    # 获取最新分类3个视频
/91porn         # 使用默认参数
```

#### 命令别名
```
/91 [分类] [数量]
/爬取视频 [分类] [数量]
```

### 返回结果格式

插件会返回以下信息：
1. 📊 统计信息（分类、数量、耗时）
2. 🖼️ 打码后的视频封面图片
3. 📹 视频详细信息（标题、链接）

示例输出：
```
✅ 用户名, 获取完成！

📊 统计信息:
   🏷️ 分类: hot
   📝 视频数: 5
   ⏱️ 耗时: 3.2秒

[打码后的封面图片1]

📹 视频 1/5
🏷️ 标题: 视频标题1
🔗 链接: https://91porn.com/view_video.php?viewkey=...

[继续显示其他视频...]
```

## 配置文件

插件配置文件位于 `_conf_schema.json`，包含以下配置项：

### 基本配置
```json
{
  "91porn_enabled": true,              // 是否启用插件
  "91porn_category": "rf",             // 默认分类
  "91porn_viewtype": "basic",          // 默认视图类型
  "91porn_max_pages": 5,               // 默认最大页数
  "91porn_workers": 3,                 // 工作线程数
  "91porn_delay": 1.0,                 // 请求延迟(秒)
  "91porn_timeout": 15.0,              // 请求超时(秒)
  "91porn_user_agent": "..."           // 用户代理
}
```

### 图片打码配置
```json
{
  "image_censorship_enabled": true,     // 是否启用图片打码
  "mosaic_level": 0.8,                 // 马赛克程度(0.1-1.0)
  "max_thumbnails": 5                  // 最大封面数量
}
```

### 网络配置
```json
{
  "proxy": "",                         // 代理设置
  "timeout": 30,                       // 网络超时
  "max_concurrent_tasks": 2            // 最大并发任务数
}
```
